# 📊 Практичні роботи з курсу «Бази даних»

Цей репозиторій містить набір виконаних практичних та лабораторних робіт, присвячених проєктуванню, створенню та управлінню реляційними базами даних (на прикладі MySQL), а також аналізу й візуалізації даних за допомогою Python.

## 📂 Структура репозиторію

Репозиторій розбитий на директорії відповідно до тем курсу:

* **`practice_3/` — Проєктування реляційних БД:**
  * Файл моделі MySQL Workbench (`db_1.mwb`) з розробленою ER-діаграмою бази даних видавництва.
* **`practice_4/` — DDL та DML команди:**
  * Скрипти для фізичного створення таблиць (`initial_db.sql`).
  * Наповнення бази даних початковими тестовими даними (`insert_db.sql`).
  * Перевірка створеної структури (`check_db.sql`).
* **`practice_5/` — DQL команди (Основи вибірки):**
  * Широкий набір скриптів (`task_1.sql` – `task_19.sql`) для вибірки даних (базові запити, фільтрація, сортування, агрегація, `JOIN` та складні додаткові завдання).
* **`practice_6/` — Складні SQL-вирази та Тригери:**
  * Реалізація бізнес-логіки на рівні бази даних за допомогою тригерів (`trigger_1.sql`, `trigger_2.sql`).
  * Скрипти для тестування коректності роботи тригерів (`check_triggers.sql`).
* **`practice_7/` — Вкладені запити та VIEW:**
  * Аналітичні запити з використанням підзапитів, CTE (тимчасових таблиць у пам'яті) та створення віртуальних таблиць (VIEW) для оптимізації роботи.
* **`practice_8/` — Вбудовані функції SQL:**
  * Робота з текстовими, числовими, часовими та логічними функціями MySQL для форматування даних «на льоту».
* **`practice_9/` — Data Analytics & ETL на Python:**
  * `analytics_base.py` — підключення до БД, завантаження «сирих» даних, очищення та розрахунок KPI (міні-ETL).
  * `analytics_visuals.py` — побудова аналітичних графіків (Matplotlib, Seaborn).
  * `app.py` — інтерактивний веб-дашборд на базі **Streamlit** для візуалізації показників продажів видавництва.
  * Папки `data/raw/` та `data/processed/` для збереження CSV-файлів на різних етапах обробки.

## 🛠 Технології та інструменти
* **СУБД:** MySQL 8.0, MySQL Workbench
* **Мова запитів:** SQL (DDL, DML, DQL, TCL)
* **Програмування:** Python 3.x
* **Бібліотеки Python:** `pandas`, `matplotlib`, `seaborn`, `sqlalchemy`, `pymysql`, `streamlit`, `python-dotenv`

## 🚀 Як запустити Python-аналітику (Practice 9)

**Примітка:** Папка з віртуальним середовищем (`env`) навмисно не відстежується в Git. Щоб запустити веб-додаток локально, виконайте наступні кроки:

### 1. Перейдіть до папки:

```bash
cd practice_9
```

---

### 2. Створіть та активуйте нове віртуальне середовище:

```bash
python -m venv env
# Для Windows:
env\Scripts\activate
# Для Mac/Linux:
source env/bin/activate
```

---

### 3. Встановіть необхідні залежності:

```bash
pip install pandas matplotlib seaborn sqlalchemy pymysql streamlit python-dotenv pyarrow
```

---

### 4. Створіть файл `.env` та вкажіть ваші дані для підключення до MySQL:

```bash
DB_HOST=localhost
DB_PORT=3306
DB_USER=root
DB_PASSWORD=ваш_пароль
DB_NAME=publishing
```

---

### 5. Запустіть Streamlit-додаток:

```bash
python -m streamlit run app.py
```
